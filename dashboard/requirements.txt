# pandas 3.0 in particular is a major version bump from what's used
# elsewhere in the project (local dev, the Lambda layer), so this pin is
# deliberately scoped to the dashboard only; SageMaker training and
# bid_consumer keep the older, Python-3.11/3.12-compatible pins since nothing
# forces them onto 3.14.
